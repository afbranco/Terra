Terra pre-built environment

Terra system was pre-installed in an Ubuntu Linux virtual machine to support teaching/learning and application development. This VM has installed the following softwares.

  • WSN OS: TinyOS and TinyOS Tools
  • Simulators: TOSSIM and Avrora
  • Terra Tools
    • TerraNet and TerraGrp specific customization
    • Text editor + Terra Compiler
    • Visual simulator
    • Program load and monitoring tool
  • Preconfigured SerialForward tool: PC <--> Mote communication tool
  • TOSSAM Lua lib: PC <--> Mote communication Lua library

The VM was exported using the OVF/OVA format (Open Virtualization Format) of VirtualBox.

Download the .ova file and import it into VirtualBox using the File menu the Import Appliance option. In VMware Workstation Player you need only to choose Open Virtual Machine in the menu options Obs.: Both the user and password are 'terra'.
